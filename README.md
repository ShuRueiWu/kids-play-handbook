## 💡 使用小提醒

- 過程比成品重要，跟著孩子當天的興趣走，玩不下去就換。
- 小物（鈕扣、豆子、串珠、小磁鐵）對仍會放嘴巴的孩子有誤食風險，請全程看顧。
- 詳細安全提醒見手冊內「🛟 居家陪玩安全小提醒」。
